# HydroForecast-MLP-River-Flow-Prediction-from-Scratch
NumPy-based MLP for river flow forecasting in Skelton, UK. Features a robust data pipeline with PCHIP interpolation, outlier removal, and time-lagged engineering. The neural network includes manual implementations of Backpropagation, Momentum, Bold Driver, and Weight Decay to ensure precise time-series regression and model optimization.
